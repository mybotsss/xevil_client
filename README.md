# xevil_client

- Download free version of xevil captcha decode tool: <https://xevil.net>
- Note: OCR decode accuracy is %90-100 for a lot of OCR types but free version is limits to %70 and resolve time is ~1 second.
- Other captcha types like `anticaptchaofficial` can be reviewed and added to this project.

## anticaptchaofficial

Ported neccessary functions from anticaptchaofficial package. There is no need to change hosts file for xcode.
Change IP address at `anticaptchaofficial.antiNetworking.make_request` function. Default is: `http://localhost/`
